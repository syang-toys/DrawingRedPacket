### 修改部分

删除一些重复代码并在目录结构上进行了一定程度的优化

## 运行

请使用微信web开发者工具打开项目文档并配置相应参数信息

- 微信小程序 AppID
- LeanCloud 程序配置参数

如果发生请求上的问题，请注意配置微信合法域名，相应链接请参考：

[小程序域名白名单配置](https://leancloud.cn/docs/weapp-domains.html)
